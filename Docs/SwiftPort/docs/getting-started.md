# Getting Started

This Swift package is a porting workspace, not the production distribution.
Build and run it with SwiftPM:

```shell
swift build
swift run codex --help
```

For the production Codex CLI walkthrough, see
https://developers.openai.com/codex/getting-started.

For Swift-port status, read `Docs/PORTING.md` before assuming a command is
complete.
